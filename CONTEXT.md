# Agent Inbox

Agent Inbox 是 agent 工作的关注与跟进上下文，用会话状态帮助用户及时处理已完成工作，而不是通用任务管理器。

## Language

**运行中会话**:
Agent 当前仍在执行工作的会话。
_Avoid_: 活跃待办、进行中任务

**待办**:
Agent 工作已结束、仍等待用户确认或继续处理的会话。
_Avoid_: 普通任务、历史记录

**新待办**:
应用已观察为运行中、随后首次观察为待办的同一会话。应用启动时直接发现的已有待办不属于新待办。
_Avoid_: 新任务、新会话

**会话源**:
会话数据的来源种类（Codex、Grok）。列表可混排多源，用标签区分。
_Avoid_: 模型名、账号、安装路径

**会话身份**:
由会话源 + 源原生会话 ID 共同标识一条会话。源原生 ID 保持各工具原格式；完成确认等持久化使用二者的复合键。
_Avoid_: 只靠文件名、只靠项目名

**Grok 待办**:
Grok 会话进程已退出，且该会话曾完成过至少一轮工作，仍等待用户确认。进程仍存活、仅一轮回复结束并等待用户输入时，不算待办。
_Avoid_: 每次模型回复结束、turn 完成
